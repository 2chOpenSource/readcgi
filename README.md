2ch Read.Cgi
============

* Vue.js
* vue-router
* material-design-lite
* webpack

などを使用してモバイル、PCどちらでも利用可能なWebベースの専用ブラウザ(like itest)を作るプロジェクトです

## ビルド方法
1. [Node.js](https://nodejs.org)をインストール
2. ```npm install -g webpack```でwebpackをインストール
3. Cloneしたディレクトリで```npm install```を実行
4. ```webpack```を実行
